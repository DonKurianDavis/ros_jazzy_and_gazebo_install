# Installation of ROS Jazzy and Gazebo Harmonic.
The file is used for automating the process of ROS Jazzy installation as well as Gazebo Harmonic. 
Download the "jazzy.sh" file, convert the file into executable and run the shell file.
```
wget https://github.com/DonKurianDavis/ros_jazzy_and_gazebo_install/blob/main/jazzy.sh
chmod +x jazzy.sh
./jazzy.sh
```
