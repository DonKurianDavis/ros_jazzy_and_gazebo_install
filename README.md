<img width="2315" height="88" alt="image" src="https://github.com/user-attachments/assets/b5295e6d-5290-47df-9845-8f1e26c8a62a" /># Installation of ROS Jazzy and Gazebo Harmonic.
The file is used for automating the process of ROS Jazzy installation as well as Gazebo Harmonic. 
The following commands is used to download the "jazzy.sh" file, convert the file into executable and run the shell file.
Copy and Paste the following commands for installation.
```
wget https://raw.githubusercontent.com/DonKurianDavis/ros_jazzy_and_gazebo_install/refs/heads/main/jazzy.sh
chmod +x jazzy.sh
./jazzy.sh
```
